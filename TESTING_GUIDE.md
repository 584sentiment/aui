```bash

# 3. 查看报告

```

## 🐛 问题排查

### 如果浏览器无法打开

```bash
# 手动启动服务器
pnpm dev

# 然后访问
open http://localhost:3000
```

### 如果端口被占用

```bash
# 查找占用端口的进程
lsof -i :3000

# 杀死进程
kill -9 <PID>
```

### 如果测试失败

1. 检查开发服务器是否运行
2. 检查浏览器控制台是否有错误
3. 查看测试截图和trace
