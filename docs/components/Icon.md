# Icon 图标

基于 [Lucide Icons](https://lucide.dev) 的美观、一致的图标工具包。

## 特点

- ✅ **1000+ 图标** - 丰富的图标集合
- ✅ **Tree-shakable** - 只打包使用的图标
- ✅ **Vue 3 原生** - 完整的 TypeScript 支持
- ✅ **高度可定制** - 颜色、大小、描边宽度
- ✅ **MIT 许可** - 免费使用

## 安装

\`\`\`bash
pnpm add lucide-vue-next
\`\`\`

## 基础用法

:::demo 基础图标用法

icon/basic

:::

## 常用图标

### 导航图标
- `ChevronDown` - 下箭头
- `ChevronUp` - 上箭头  
- `ChevronLeft` - 左箭头
- `ChevronRight` - 右箭头
- `Menu` - 菜单
- `Home` - 首页

### 操作图标
- `Edit` - 编辑
- `Trash` - 删除
- `Search` - 搜索
- `Plus` - 添加
- `Minus` - 减少

### 状态图标
- `CheckCircle` - 成功
- `XCircle` - 错误
- `AlertTriangle` - 警告
- `Info` - 信息
